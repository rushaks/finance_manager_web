Finance Manager PWA

Персональный финансовый менеджер — простое и удобное приложение для управления личными финансами с поддержкой офлайн-режима.


📱 О приложении
Finance Manager — это прогрессивное веб-приложение (PWA) для учета доходов и расходов. Приложение работает полностью в офлайн-режиме, все данные хранятся локально в вашем браузере. Никаких серверов, никаких облачных хранилищ — только ваши данные на вашем устройстве.

🎯 Ключевые особенности

📊 Управление счетами — создавайте неограниченное количество счетов с разными валютами  
💰 Учет транзакций — доходы, расходы и переводы между счетами  
📈 Аналитика — наглядная статистика с графиками и фильтрами  
📱 PWA поддержка — установите приложение на главный экран смартфона  
🌐 Офлайн-режим — приложение работает без интернета  
🔒 Приватность — все данные хранятся локально на вашем устройстве  

✨ Возможности

🎯 Особенности реализации  
📦 Локальное хранилище  
Все данные хранятся в SharedPreferences с кэшированием в памяти для быстрой работы. Приложение использует паттерн "write-through" для синхронизации кэша и диска.

🔄 Автоматические обновления  
Приложение автоматически обновляет данные при изменении (добавление, редактирование, удаление транзакций/счетов/категорий).  

📊 Оптимизация производительности  
Кэширование данных диаграмм  
Пагинация для списка транзакций (по 100 записей)  
Использование RepaintBoundary для сложных виджетов  
Оптимизированная сортировка данных  

🌐 Офлайн-режим
Все данные доступны без интернета  
Service Worker кэширует все ресурсы  
Автоматическая проверка обновлений
